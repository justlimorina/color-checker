User Guide & Glossary

1. Color Formats

How color is represented in code.

  • HEX: 6-digit code (e.g. #6750A4). Standard for web design.
  • RGB: Red, Green, Blue (0-255). Used for screen displays.
  • HSL: Hue (color), Saturation (intensity), Lightness (brightness). Great for designers.
  • CMYK: Cyan, Magenta, Yellow, Black (0-100%). Used for physical printing.

2. Color Variations

  • Tints: Colors mixed with White. Increases light.
  • Shades: Colors mixed with Black. Decreases light.
  • Tones: Colors mixed with Grey. Mutes the color.

3. Accessibility (WCAG)

WCAG ensures colors are readable for everyone, including those with visual impairments.

  • Contrast Ratio: Brightness difference. Higher is better.
  • AA: Standard compliance (4.5:1 ratio). Recommended for all websites.
  • AAA: Enhanced compliance (7:1 ratio). Highest accessibility tier.

4. Color Harmonies

Predefined rules for picking colors that look good together based on the color wheel.
